

## 🎮 CS2ModForge

> 🔧 **Modding toolkit for Counter-Strike 2** — lightweight, scriptable, and community-driven.

---

## 🚀 About

**CS2ModForge** is a modular framework for creating, testing, and sharing mods for Counter-Strike 2.  
It provides a clean API, built‑in asset pipeline, and hot‑reload support — so you can focus on ideas, not boilerplate.


⚙️  Zero‑dependency core
🧩  Plugin‑based architecture
📦  Ready‑to‑use mod templates
🔄  Live‑reload during development


---

## ✨ Features

- 🧠 **Simple scripting** — Lua & JSON based mod definitions  
- 🗂️ **Asset packer** — bundle textures, sounds, and models  
- 🧪 **Sandbox mode** — test mods without risking your main client  
- 📊 **Performance profiler** — see exactly what your mod affects  
- 🔌 **Community plugin registry** — share and discover mods

---

## 📦 Quick Start

```bash
git clone https://github.com/cs2-mod-forge.git
cd cs2-mod-forge
./setup.sh
./forge create my_first_mod
./forge build
```

> 🧪 **Requires:** CS2 (Steam) + .NET 8.0 SDK
